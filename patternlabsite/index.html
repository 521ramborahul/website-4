What is Pattern Lab

Features

At its core, Pattern Lab is a tool that helps create frontend UI patterns and work with dynamic content.

  • Nested Includes - Russian nesting dolls
  • Dynamic Data - Pattern Lab helps you dynamically swap in content
  • Pattern Documentation
  • Viewport resizer tool
  • Annotations
  • Language agnostic - flexible, use whatever tools you want
  • Pattern starter kits - available Bootstrap, Foundation, Material Design
  • Extensible - Choose your own templating engine, or build a plugin to cater Pattern Lab to your specific needs.
