Pattern Lab

Create design systems with atomic design

Pattern Lab is a collection of tools to help create and maintain atomic design systems.

See It In Action View On Github

What

Pattern Lab is a series of tools to create your very own custom atomic design system. Here's the general gist:

  1. Atoms

  2. Molecules

  3. Organisms

  4. Templates

  5. Pages

What Pattern Lab Is

  • A static site generator — At its core, Pattern Lab is simply a way to construct an interface by stitching atoms, molecules, and organisms together to form templates and pages.
  • A comprehensive component library — Pattern Lab serves as your project's pattern library and frontend style guide. But it goes further as you're able to simulaneously see modules abstractly and in context.
  • A pattern starter kit — Pattern Lab provides a whole slew of patterns to get started. Don't use them all. You should use Pattern Lab to establish your own atoms, molecules, organisms, templates, and pages. Pattern Lab lets you write your HTML, CSS, and Javascript however you want.
  • A Viewport Resizer — It's our job to make sure our designs look and function beautifully across a whole slew of Web-enabled devices, even those that haven't been imagined yet. Pattern Lab provides a robust viewport resizing tool to helps you create great designs across all screen sizes, and also serves as a great educational tool for clients and team members.
  • An Annotation Tool (coming soon) — Pattern Lab provides a way to annotate patterns in order to communicate how the bits of your interface

What Pattern Lab Isn't

  • A UI framework — Pattern Lab isn't Bootstrap. It's not Foundation. It's not flat. It's not skeumorphic. Pattern Lab provides the plumbing for creating design systems. How it looks and how it's coded is entirely up to you.
  • Language-dependent — Like Sass? Cool, go nuts with it. Like Less instead? That's cool too. Don't like any preprocessors at all? Totally fine. How about jQuery? Gotta have it or wouldn't touch it with a 10 foot pole? Pattern Lab lets you write HTML, CSS and JavaScript however want. Write code as awesome or as crappy as you like. Pattern Lab simply provides a scaffolding with which to stitch together your interface.
  • Incredibly Rigid — Pattern Lab is designed to be flexible with regards to naming and structure. There's really no One Right Way to do this, and we think that's just fine.

Why

Tiny Bootstraps, for Every Client

Dave Rupert

This style guide keeps track of every module going onto the site, creates a shared vocabulary between everyone involved in the project, makes testing easier, establishes a better workflow, and builds a canonical reference to keep coming back to.

Who

Pattern Lab benefits everyone involved in a web project. It can serve as a glue that holds a project together.

  • Information Designers—Designing static sketches and wireframes for the multi-device Web can be challenging. Pattern Lab helps integrate modular thinking into the design process right out of the gate, and helps demonstrate responsive patterns to create.
  • Visual Designers—Visual designers can use Pattern Lab as a way to see how each module works in a responsive environment and also to keep track of all the design's raw materials.
  • Developers—Pattern Lab provides developers with an efficient system for quickly developing prototypes and final front-end code.
  • Clients and managers— can use it as a way of keeping track of everything going on the site and as an educational resource.

How

You can learn more about how to get started with Pattern Lab by viewing the documentation.

Where

Pattern Lab should live in an environment accessible via a URL, which ensures everyone involved with the project can view the design system. Optionally (and ideally) your Pattern Lab can be published publicly like other style guides created by Starbucks, The BBC, and The Guardian.
